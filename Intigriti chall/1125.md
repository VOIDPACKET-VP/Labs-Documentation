# Intigriti Monthly Challenge 1125 - SSTI & JWT Bypass

**Date**: 2025-18-11
**Platform**: Intigriti
**Difficulty**: Medium
**Vulnerabilities**: JWT Authentication Bypass, Server-Side Template Injection (SSTI)

## Discovery Timeline
- **JWT Analysis**: Identified unsigned token acceptance
- **Admin Access**: Modified JWT to gain admin privileges  
- **SSTI Detection**: Found template injection in display name field
- **RCE Achievement**: Executed system commands via Jinja2
- **Flag Extraction**: Used grep to locate the flag

## Key Techniques Used
- JWT token manipulation without secret
- Jinja2 SSTI exploitation
- Python object traversal (self.__init__.__globals__)
- System command execution through template injection

## Lessons Learned
- Always verify JWT token signatures
- Sanitize all user inputs in template rendering
- Context-aware escaping is crucial for template engines
- grep is powerful for CTF flag hunting
